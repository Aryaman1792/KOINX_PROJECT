# Tax Loss Harvesting Calculator

A modern, highly-interactive React web application built with Next.js and Tailwind CSS that helps users optimize their tax liabilities by simulating tax loss harvesting strategies in real-time.

## Features

- **Real-Time Tax Computations**: Instantly calculate Net Short-Term and Long-Term Capital Gains.
- **Dynamic Harvesting**: Interactively select/deselect cryptocurrency holdings to see the immediate effect on your effective capital gains and overall tax savings.
- **Responsive Layout**: Designed to work beautifully on both desktop and mobile devices.
- **Dark Theme**: Modern UI styled accurately to match premium fintech dashboards.

## Tech Stack

- Next.js (App Router)
- React
- Tailwind CSS v4
- Lucide React (Icons)
- TypeScript

## Setup Instructions

1. Clone the repository:
   ```bash
   git clone https://github.com/Aryaman1792/KOINX_PROJECT.git
   cd KOINX
   ```

2. Install the dependencies:
   ```bash
   npm install
   ```

3. Run the development server:
   ```bash
   npm run dev
   ```

4. Open [http://localhost:3000](http://localhost:3000) with your browser to see the application.
